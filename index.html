Cara Upload Foto & MP3 ke GitHub

⚠ Kesalahan Umum: Link seperti github.com/user/repo/blob/main/file.mp3 adalah halaman web, BUKAN file asli! Browser tidak bisa memutarnya.

Link yang Benar (Raw):

https://github.com/user/repo/blob/main/file.mp3 https://raw.githubusercontent.com/user/repo/main/file.mp3

Cara Mendapatkan Raw Link:

  1. Upload file (gambar JPG/PNG atau MP3) ke repository GitHub.
  2. Klik file di GitHub, lalu klik tombol "Raw" di pojok kanan atas.
  3. Copy URL — harus dimulai dengan raw.githubusercontent.com.
  4. Paste ke kode di field coverImage atau imageUrl atau audioUrl.

Contoh Menambahkan Foto:

{ id: 1, title: "Violin Sound", instrument: "Piano", imageUrl: "https://raw.githubusercontent.com/USER/REPO/main/violin.jpg", audioUrl: "https://raw.githubusercontent.com/USER/REPO/main/violin.mp3" }

Format Gambar yang Didukung:

  • JPG / JPEG — paling direkomendasikan (ukuran kecil)
  • PNG — kualitas tinggi tapi file lebih besar
  • WebP — ukuran paling kecil, didukung browser modern
  • Batasan GitHub: 25 MB per file

Tips Ukuran Foto:

  • Cover buku: 200x260 px sudah cukup
  • Foto halaman: 400x300 px sudah cukup
  • Perkecil foto sebelum upload agar loading cepat
