Cara Upload MP3 & Mengatur Link

Karena ini adalah file HTML statis, Anda tidak bisa "upload" file langsung ke sini seperti di Instagram. Anda harus menghosting file MP3 Anda terlebih dahulu.


Langkah-langkah:

  1. Siapkan File MP3: Pastikan nama file unik (contoh: piano-buku1.mp3).
  2. Upload ke Cloud Hosting: Gunakan layanan seperti Google Drive (set ke publik), Dropbox, atau hosting cPanel Anda.
  3. Copy Link Langsung (Direct Link): Pastikan link berakhiran .mp3, bukan halaman preview.
  4. Edit Kode: Buka file HTML ini, cari bagian JavaScript di bawah, dan ganti link dummy di dalam variabel booksData.
  5. Buat QR Code: Gunakan link website + parameter.
    Contoh: websiteanda.com?book=1&page=2

Contoh Struktur Data di Kode:

{ id: 1, title: "Buku Mewarnai Vol. 1", pages: [ { id: 1, title: "Piano", audio: "LINK_MP3_DISINI" }, { id: 2, title: "Gitar", audio: "LINK_MP3_DISINI" } ] }
